# How to Install
## Windows
1. Go to the [MRCube page on the Microsoft Store](https://apps.microsoft.com/detail/9nmgtwtl1fvg?hl=en-US) and click the "View in Microsoft Store" button.
2. Click the "Install" button.
3. Run the shortcut created on your desktop.

## macOS (dmg file)
1. Download and extract **mrcube-26.4.3.dmg** from the [download page](https://github.com/mr-3/mrcube/releases) and copy **MRCube.app** to the 'Applications' folder.
2. Run **MRCube.app**.
3. When the message '"MRCube" is an application downloaded from the Internet. Are you sure you want to open it?' appears, click the "Open" button.

## macOS (Homebrew)
1. Install [Homebrew](https://brew.sh/).
2. Run ``brew tap mr-3/tap`` and ``brew install --cask mrcube`` in Terminal.app.
3. Launch "MRCube.app" from your Applications directory.
4. When the message '"MRCube" is an application downloaded from the Internet. Are you sure you want to open it?' appears, click the "Open" button.
